# Output Templates

Use these templates to keep `tushare-data` outputs concise, decision-useful, and human-readable.

General rule:
- conclusion first
- scope and time range second
- key numbers third
- caveats fourth
- file paths last when files were exported

Do not dump raw field names unless the user explicitly asks.

---

## 1. Single-stock quick view

### Best for
- 看下 XX 最近怎么样
- 这票最近强不强
- 给我快速看看 XX

### Template

```md
## 一句话结论
<用一句人话概括这只股票最近的状态>

## 区间与口径
- 标的：<ts_code / 名称>
- 时间范围：<start> ~ <end>
- 数据口径：<日线 / 复权 / 含哪些基础指标>

## 关键数据
- 区间涨跌幅：...
- 最新收盘价：...
- 区间高 / 低点：...
- 成交额 / 换手率变化：...

## 简短解读
- <趋势特征>
- <活跃度 / 波动特征>
- <是否强于板块/指数，如有>

## 注意事项
- <数据限制 / 非交易日 / 复权口径说明>
```

## 6. Fund-flow summary

When doing northbound / southbound style tasks, clearly distinguish:
- overall direction / flow trend
- active traded names
- stable net-inflow fields vs unstable or missing fields

### Best for
- 北向资金最近在买什么
- 主力资金流向如何
- 哪个板块最吸金

### Template

```md
## 一句话结论
<资金更偏向哪里，先说方向>

## 范围与口径
- 资金口径：<北向 / 主力 / 板块 / 龙虎榜>
- 时间范围：<start> ~ <end>
- 样本范围：<全市场 / 某板块 / 某指数成分>

## 核心结果
- 净流入最多：...
- 净流出最多：...
- 持续性较强的方向：...

## 关键表
| 标的/板块 | 净流入 | 区间表现 | 备注 |
|-----------|--------|----------|------|
| ...       | ...    | ...      | ...  |

## 简短解读
- <资金是否与价格共振>
- <是否只是单日噪声>
- <哪些方向值得继续跟踪>

## 注意事项
- <资金口径差异 / 时间窗影响>
```

## 11. Research brief

- 给我做个简报
- 给个结论版分析

### Template

```md
## 一句话结论
<先用一句话概括这个标的/板块当前最值得注意的点>

## 1. 行情表现
- <区间涨跌、强弱、活跃度>

## 2. 财务趋势
- <营收、利润、质量变化>

## 3. 估值水平
- <贵不贵、是否有支撑>

## 4. 资金流情况
- <是否有持续关注>

## 5. 公告 / 新闻 / 催化
- <最近最重要的 2～3 条>

## 6. 风险点
- <最值得警惕的点>

## 7. 值得继续深挖的问题
- <下一步若继续研究，应该补什么>
```

## Quick rule

好的输出，不是字段更全。
而是：
- 先给判断
- 再给证据
- 说明口径
- 告知限制
- 如果导出了文件，明确路径
