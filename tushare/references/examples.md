# Examples

This file contains natural-language examples for `tushare-data`.
Use these examples to recognize task intent, choose default workflows, and keep outputs human-readable.

---

## 1. 单标的行情

### 用户说法
- 看下宁德时代最近三个月走势
- 茅台今年以来涨了多少
- 招行这两年最大回撤大概多少
- 帮我看下中际旭创最近强不强
- 这只票最近有没有放量

### 默认处理思路
- 解析标的
- 默认取近 20 个交易日 / 近 3 个月 / 用户指定区间
- 拉行情 + 必要的 `daily_basic`
- 输出区间涨跌、活跃度、高低点、波动或回撤

---

## 2. 多标的对比

### 用户说法
- 比一下茅台、五粮液、泸州老窖近一年的涨幅和估值
- 对比一下沪深300、中证500、创业板今年表现
- 把几只光伏龙头最近 60 天走势放一起看
- 宁德时代和比亚迪谁最近更强

### 默认处理思路
- 锁定对象
- 统一时间范围
- 选择 3～5 个关键指标
- 输出对比表 + 总结“谁在哪方面更强”

---

## 3. 财务趋势

### 用户说法
- 看下比亚迪最近 8 个季度营收和净利润趋势
- 宁德时代最近几个季度利润怎么样
- 最近业绩是变好还是变差
- 帮我看下这家公司财务质量怎么样

### 默认处理思路
- 默认最近 8 个季度 + 最近年度
- 拉 `fina_indicator` + `income` + 必要现金流/资产负债数据
- 输出营收、净利润、毛利率、ROE、现金流质量的趋势总结

---

## 4. 估值与基本面

### 用户说法
- 茅台现在估值算高吗
- 现在银行股里谁 PE/PB 最低
- 帮我筛一批低估值高股息股票
- 查下这家公司最新市值、PE、PB、股息率

### 默认处理思路
- 使用 `daily_basic` 为主
- 必要时结合 `fina_indicator`
- 输出估值水平、横向对比、筛选条件与结果

---

## 5. 财务筛选 / 条件选股

### 用户说法
- 找高 ROE 低负债的公司
- 哪些公司利润增长最快
- 找最近业绩预增但涨幅不大的票
- 帮我筛一下现金流好、毛利率稳定的股票

### 默认处理思路
- 先定义筛选池
- 再定义时间口径
- 拉核心财务和估值指标
- 过滤、排序、输出 Top N
- 明确筛选条件

---

## 6. 资金流

### 用户说法
- 北向资金最近在买什么
- 今天北向资金流入最多的股票有哪些
- 最近一周主力资金持续流入的票有哪些
- 这票最近资金是流入还是流出
- 哪个板块最近最吸金

### 默认处理思路
- 明确资金口径：北向 / 主力 / 板块 / 龙虎榜
- 默认时间窗近 5～20 个交易日
- 联动价格表现解释，不把单日噪声说成趋势

## 7. 龙虎榜 / 情绪 / 打板

### 用户说法
- 今天市场情绪怎么样
- 最近连板最高的是谁
- 今天涨停梯队怎么样
- 最近炸板率高吗
- 哪些票最近游资最活跃

### 默认处理思路
- 选 `limit_list_d`、`limit_step`、`top_list`、`kpl_list` 等
- 输出情绪温度、连板结构、热点方向、代表标的

## 8. 板块 / 行业 / 概念

### 用户说法
- 最近哪个板块最强
- 半导体板块最近一个月强不强
- 机器人为什么涨
- 哪些主题最近最热
- 这个概念板块里有哪些成分股

### 默认处理思路

- 拉板块区间表现
- 必要时联动成分股、资金流、消息面

## 9. 公告 / 新闻 / 研报 / 政策

### 用户说法
- 帮我梳理下寒武纪最近的重要公告
- 最近机器人板块有什么消息面催化
- 最近有什么重要政策
- 给我拉一份最近的财经快讯摘要
- 最近关于 AI 算力的研报有哪些

### 默认处理思路
- 先定对象与时间窗
- 拉公告 / 新闻 / 研报 / 政策数据
- 去噪，提炼主线
- 区分事实与媒体解读

## 10. 宏观

### 用户说法
- 最近宏观环境怎么样
- 看一下最近 CPI、PPI、PMI 变化
- 当前市场风格偏成长还是价值
- 最近流动性松不松
- 宏观对 A 股偏多还是偏空

### 默认处理思路
- 拉最近 6～12 期关键指标
- 比较最新值、前值、同比趋势
- 轻量关联到市场风格，不做过度因果化

## 14. 高频指标缩写与时间表达

### 用户说法
- 帮我筛一批高 ROE 低负债的股票
- 看下茅台 PB 和 PE 现在处于什么水平
- 最近 8 个季度净利润趋势怎么样
- 最近 8 个季度营收和净利润趋势怎么样
- 帮我看下最近 20 个交易日谁最抗跌
- 哪些公司 YoY 改善最明显
- 看下 TTM 口径下谁更便宜
- 最近 12 个月谁涨得最好
- 帮我找近 5 年现金流比较稳的公司

### 默认处理思路
- 把 ROE / PE / PB / YoY / TTM 视为高频正常表达
- 把“8 个季度 / 20 个交易日 / 12 个月 / 近 5 年”视为明确时间范围
- 如果口径会显著影响结论，要在输出里点明

## 15. 超短高频问法

### 用户说法
- 北向最近买什么
- 北向资金最近在买什么
- 今天最强板块
- 这票值不值得看
- 哪些票最抗跌
- 最近谁最强
- 这票最近有资金关注吗
- 最近有没有催化
- 给我拉一份数据

### 默认处理思路
- 不因用户说得短就降级成模糊意图
- 先按高频研究语境理解：
  - 北向 → 资金流
  - 最强板块 → 板块轮动
  - 值不值得看 → 快速研究简报
  - 最抗跌 → 排行/筛选
  - 资金关注 → 资金流 / 活跃度
  - 催化 → 公告 / 新闻 / 政策
  - 拉一份数据 → 导出 / 研究准备

---

## Quick rule

当用户的说法像“看下 / 查下 / 比一下 / 筛一下 / 梳理一下 / 导出一下 / 快速研究一下”时，优先识别：
- 是查行情
- 看财务
- 做对比
- 做筛选
- 看资金流
- 看板块
- 梳理事件
- 看宏观
- 还是做导出/研究准备

不要先退回到接口名思维。
